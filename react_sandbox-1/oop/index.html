Objektorientierte Programmierung

OoP ist ein Programmierparadigma (Programmierstil) basierend auf das Konzept von Objekten.


Was ist in objektorientierter Programmierung eine Methode?

Ein Objekt ist eine Art Container. Dessen Inhalt, seine Eigenschaften werden durch properties beschrieben. Properties enthalten values/Werte und keys/Schlüsselnamen, zu denen die values zugeordnet werden. Ein Objekt kann auch Methoden enthalten. Eine Methode ist dazu da, um zu definieren, was für eine Aktion ausgelöst werden soll. Es definiert die Aktion in Form einer function, die dann aufgerufen wird.

Was ist OoP eine Eigenschaft/Attribut/Property? Was hat das mit Klassen zu tun? Was ist überhaupt eine Klasse?

In OoP sind Klassen(classes) eine Art templates für Javascript objects, sozusagen so etwas wie eine Vorlage und geben vor, wie das object, um das es geht, allgemein aussehen soll, das heißt, welche Eigenschaften, auch Property oder Attribut genannt, es besitzen soll. Properties sind wichtig, weil dadurch ein object individueller gemacht wird. Zum Beispiel ein Hund ist ein Tier/animal, animal wäre die class. Diese class beschreibt, dass alle Objekte, die von dieser Klasse gebildet werden, vier Beine haben soll. Das heißt, das Object hat als Property vier Beine -> {legs: 4}

Was ist eine Object-Instanz?

Ein object ist eine Instanz von einer class. Im obigen Beispiel ist animal die class und dog(Hund) ist eine Instanz davon.

Ein constructor ist eine Methode in einer class und definiert die properties, die jede instanz, die aus dem object aus dieser class entsteht haben soll. Im obigen Beispiel würde es heißen, dass jede Instanz der class "Animal" Beine haben soll. Die genaue Anzahl wird noch nicht festgelegt. (Es wäre zwar möglich, sie schon in der class festzulegen, aber das ist nicht sinnvoll.) Erst wenn die Instanz mit einer neuen Variablen erstellt wird, werden die genauen Werte mit übergeben und in dieser Instanz gespeichert, indem die class aufgerufen wird. Man erstellt dann zum Beispiel zu der class animal eine Instanz mit dem Namen dog in dieser Form: const dog = new Animal ("4"). Das object dog hat dann als property legs:4. Die Verwendung von "new" ist dabei notwendig, wenn die Instanz von der class gebildet werden soll. Beispiel als code folgt weiter unten.
